Http Methods Path Description
GET /users Display all users.
GET /users/:id Display a single user
PUT /users/:id Update single user record
PATCH /users/:id To modify a user's record.
POST /users Add new user to table
DELETE /users/:id To delete a single user.
GET /products Display all products
GET /products/:id Display single product
PUT /products/:id Update a single products
DELETE /products/:id Delete a product.
POST /products Add product
